# Express TypeScript Starter

Load images from URL with the ability to request an image thumbnail with a custom size,

**Technologies used:**

- Nodejs
- Expressjs
- TypeScript
- apicache
- Jasmine
- Helmet
- Custom Logger

## Instructions

### Install

```
npm install
```

### Run Server (Build & Start)

```
npm start
```

### Development (Auto-Reload)

```
npm run dev
```

### Testing

```
npm run test
```
